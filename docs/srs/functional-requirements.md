# 3 功能需求（需求必须编号，每项需求内参考作业文档中基本描述、相关功能的结构，并根据需要增减）

## 3.1 企业

**FR-101 企业信息修改**

企业用户可修改本企业的基础信息。修改完成后需保存并上报至省局备案。企业信息需按照统一模板填写，部分字段不可修改。

**数据说明：**

| 数据项       | 必填 | 说明                                          |
| :----------- | :--- | :-------------------------------------------- |
| 所属地区     | 是   | 显示企业所属地市、市县、区域，不可修改        |
| 组织机构代码 | 是   | 只可输入字母、数字，不超过9位（统一编码规范） |
| 企业名称     | 是   | 中文、英文                                    |
| 企业性质     | 是   | 两级下拉选择                                  |
| 所属行业     | 是   | 两级下拉选择                                  |
| 主要经营业务 | 是   | 按实际情况填写企业主要经营的业务              |
| 联系人       | 是   | 中文、英文                                    |
| 联系地址     | 是   | 两级下拉选择                                  |
| 邮政编码     | 是   | 只可填写6位数字                               |
| 联系电话     | 是   | 格式必须符合（区号）+电话号码或者为手机号码   |
| 传真         | 是   | 格式必须符合（区号）+电话                     |
| EMAIL        | 否   | 格式必须符合xxx@xxx.xxx                       |

**相关功能：**

- 查看当前企业信息
- 编辑可修改字段
- 保存修改并触发备案流程
- 字段格式校验

**FR-102 备案上报**

企业完成信息填写或修改后，可将备案信息上报至省局。上报后信息进入待审核状态。

**相关功能：**

- 提交备案信息
- 提示提交成功
- 查看备案状态（待审核/已通过/已退回）
- 退回时可查看退回理由

**FR-103 数据填报**

企业在规定的上报期内，填报本企业的就业失业数据。填报内容需符合模板要求，若调查期就业人数低于建档期人数，则需填写减少类型及原因。

**数据说明：**

| 数据项           | 必填 | 说明                                                         |
| :--------------- | :--- | :----------------------------------------------------------- |
| 建档期就业人数   | 是   | 填写初次建档时监测点就业人数                                 |
| 调查期就业人数   | 是   | 填写本次调查期当时的监测点就业人数                           |
| 其他原因         | 是   | -                                                            |
| 就业人数减少类型 | 否*  | 关闭破产、停业整顿、经济性裁员、业务转移、自然减员、正常解除或终止劳动合同、国际因素变化影响、自然灾害、重大事件影响、其他 |
| 主要原因         | 否*  | 产业结构调整，重大技术改革，节能减排、淘汰落后产能，订单不足，原材料涨价，工资、社保等用工成本上升，自然减员，经营资金困难，税收政策变化，季节性用工，其他，自行离职，工作调动、企业内部调剂，劳动关系转移、劳务派遣 |
| 主要原因说明     | 否*  | -                                                            |
| 次要原因         | 否   | -                                                            |
| 次要原因说明     | 否   | -                                                            |
| 第三原因         | 否   | -                                                            |
| 第三原因说明     | 否   | -                                                            |

*注：当调查期就业人数小于建档期就业人数时，就业人数减少类型、主要原因及主要原因说明为必填*

**相关功能：**

- 填报当期数据
- 保存草稿
- 提交上报
- 校验必填项及逻辑规则（数据对比、格式校验）
- 显示填报期限提醒

**FR-104 数据查询**

企业用户可查询历史填报数据及其状态（如已上报、已审核、已退回等）。查询结果仅支持浏览，不支持导出。

**相关功能：**

- 按调查期查询
- 按数据状态筛选（草稿/已上报/已审核/已退回）
- 查看数据详情
- 查看审核状态及退回备注

**FR-105 密码修改**

企业用户可修改登录密码。修改后需重新登录。

**相关功能：**

- 验证旧密码
- 输入新密码并确认（密码复杂度校验）
- 提示修改成功

## 3.2 省

### FR-201 企业备案

#### 1. 功能描述

省级用户可查看全省各市已备案企业的详细信息，并对企业备案状态进行审核。审核通过后，企业方可进行数据填报。

#### 2. 前置条件

- 用户已登录系统并具备省级角色权限
- 企业已完成基础信息填报并提交备案

#### 3. 后置条件

- 企业备案状态更新为“已备案”或“已退回”
- 已备案企业可参与后续数据填报流程

#### 4. 基本流程

- 省级用户进入“企业备案”功能模块

- 系统展示企业备案列表，默认按提交时间倒序排列

- 用户可通过调查期、地区、企业名称、备案状态等条件查询企业

- 用户点击企业名称查看详细备案信息

- 用户对备案信息进行审核

  - 若信息无误，点击“审核通过”

  - 若信息有误，点击“退回”，并填写退回原因

- 系统保存审核结果并更新备案状态

#### 5. 扩展流程

- 用户可将当前查询结果导出为Excel文件
- 支持批量审核操作（可选）

#### 6. 数据项说明

| 数据项       | 说明                                   |
| :----------- | :------------------------------------- |
| 所属地区     | 显示企业所属地市、市县、区域，不可修改 |
| 组织机构代码 | 字母/数字，不超过9位                   |
| 企业名称     | 中文/英文                              |
| 企业性质     | 两级下拉选择                           |
| 所属行业     | 两级下拉选择                           |
| 联系人       | 中文/英文                              |
| 联系电话     | 符合（区号）+电话或手机号格式          |
| 备案状态     | 待审核/已备案/已退回                   |
| 审核人       | 系统自动记录当前操作人                 |
| 审核时间     | 系统自动记录                           |

#### 7. 界面要求

- 列表页支持分页、排序、条件筛选
- 详情页只读展示企业备案信息
- 审核操作需二次确认

### FR-202 企业查询

#### 1. 功能描述

省级用户可根据需要对备案企业进行多条件查询，便于管理与统计。

#### 2. 前置条件

- 用户已登录并具备省级角色权限

#### 3. 后置条件

- 无

#### 4. 基本流程

1. 省级用户进入“企业查询”功能模块
2. 输入或选择查询条件（支持组合查询）
3. 点击“查询”，系统展示符合条件的企列表
4. 用户可点击企业名称查看详细信息
5. 点击“清除”可清空所有查询条件

#### 5. 扩展流程

- 支持导出查询结果为Excel文件

#### 6. 查询条件说明

| 查询项       | 说明         |
| :----------- | :----------- |
| 企业名称     | 模糊匹配     |
| 组织机构代码 | 精确匹配     |
| 所属地市     | 下拉选择     |
| 所属市县     | 下拉选择     |
| 企业性质     | 下拉选择     |
| 所属行业     | 下拉选择     |
| 备案状态     | 下拉选择     |
| 调查期       | 日期范围选择 |

### FR-203 报表管理

#### 1. 功能描述

省级用户对各市审核通过的企业上报数据进行最终审核，并汇总上报至部级单位。

#### 2. 前置条件

- 企业数据已通过市级审核
- 用户具备省级审核权限

#### 3. 后置条件

- 数据状态更新为“省级审核通过”或“已退回”
- 已上报数据不可再次修改

#### 4. 基本流程

1. 省级用户进入“报表管理”模块
2. 系统展示待审核的企业上报数据列表
3. 用户点击企业名称查看详细数据及报表
4. 用户进行审核操作：
   - 数据无误：点击“审核通过”
   - 数据有误：点击“退回修改”，填写退回原因
5. 系统更新数据状态
6. 审核通过后，用户可选择“上报至部委”

#### 5. 扩展流程

- 支持批量审核、批量退回
- 上报操作需二次确认

#### 6. 数据状态说明

| 状态         | 说明                     |
| :----------- | :----------------------- |
| 待省级审核   | 企业数据已通过市级审核   |
| 省级审核通过 | 省级审核通过，可上报部委 |
| 已退回       | 退回至市级或企业         |
| 已上报       | 数据已上报至部委         |

### FR-204 数据修改

#### 1. 功能描述

对于有明显错误的企业上报数据，省级用户有权进行修正。修正后的数据需与原数据区分存储，并记录修改日志。

#### 2. 前置条件

- 数据状态为“待省级审核”或“省级审核通过”
- 用户具备数据修改权限

#### 3. 后置条件

- 生成修改后的数据副本
- 原始数据保持不变
- 系统记录修改日志

#### 4. 基本流程

1. 省级用户进入“数据修改”模块
2. 按调查期、企业名称等条件定位需修改的数据
3. 查看原始数据，点击“修改”
4. 在数据填报界面进行修改
5. 保存修改内容
6. 系统生成修改记录

#### 5. 扩展流程

- 支持查看历史修改记录
- 支持导出修改日志

#### 6. 日志记录字段

| 字段       | 说明                 |
| :--------- | :------------------- |
| 修改人     | 当前操作人           |
| 修改时间   | 系统自动记录         |
| 修改前内容 | JSON格式或字段级对比 |
| 修改后内容 | JSON格式或字段级对比 |

### FR-205 数据删除

#### 1. 功能描述

省级用户可删除历史数据，适用于数据清理或错误数据移除场景。

#### 2. 前置条件

- 数据状态为“已上报”或“历史数据”
- 用户具备数据删除权限

#### 3. 后置条件

- 数据从系统中移除
- 系统记录删除日志

#### 4. 基本流程

1. 省级用户进入“数据删除”模块
2. 按调查期、企业名称等条件选择需删除的数据
3. 点击“删除”，系统弹出二次确认框
4. 用户确认后，数据被移除
5. 系统记录删除操作

#### 5. 扩展流程

- 支持批量删除
- 删除前可预览数据内容

#### 6. 删除日志字段

| 字段     | 说明             |
| :------- | :--------------- |
| 删除人   | 当前操作人       |
| 删除时间 | 系统自动记录     |
| 删除数据 | 数据摘要或ID列表 |

### FR-206 数据退回

#### 1. 功能描述

对于不符合要求的企业上报数据，省级用户可将其退回至上一级（市级或企业），并附上退回原因。

#### 2. 前置条件

- 数据状态为“待省级审核”
- 用户具备退回权限

#### 3. 后置条件

- 数据状态更新为“已退回”
- 企业端可见退回原因并可重新填报

#### 4. 基本流程

1. 省级用户进入“数据退回”模块
2. 选择一条或多条待审核数据
3. 点击“退回”，弹出退回说明填写框
4. 填写退回原因（必填）
5. 确认退回，系统更新数据状态
6. 系统记录退回日志

#### 5. 扩展流程

- 支持批量退回
- 退回原因支持模板选择

#### 6. 退回日志字段

| 字段     | 说明         |
| :------- | :----------- |
| 退回人   | 当前操作人   |
| 退回时间 | 系统自动记录 |
| 退回原因 | 用户填写内容 |
| 退回数据 | 数据ID列表   |

## 3.3 市

**FR-301 数据审核**

市级用户审核辖区内企业上报的就业失业数据。审核通过后数据可上报至省级。

**数据说明：**

审核内容为企业填报数据（详见FR-103数据说明）

**相关功能：**

- 列表展示待审核数据（分页显示，可按企业名称、调查期筛选）
- 查看企业填报详情（包含历史填报记录）
- 审核通过/退回（退回时需填写退回理由）
- 批量审核功能
- 查看审核记录（审核人、审核时间、审核结果）

**FR-302 通知发布**

市级用户可向辖区内企业发布通知。通知内容需包含标题、正文、发布时间等信息。

**数据说明：**

| 数据项   | 必填 | 说明         |
| :------- | :--- | :----------- |
| 通知标题 | 是   | 50字以内     |
| 通知内容 | 是   | 2000字以内   |
| 发布时间 | 否   | 系统自动生成 |
| 发布单位 | 否   | 系统自动生成 |

**相关功能：**

- 新增通知（填写标题、内容）
- 修改通知（仅可修改未过期的通知）
- 删除通知
- 查看已发布通知列表（按发布时间倒序）
- 通知阅读统计（可选）

**FR-303 密码修改**

市级用户可修改登录密码。修改后需重新登录。

**相关功能：**

- 验证旧密码
- 输入新密码并确认（密码复杂度校验：长度、字符类型）
- 提示修改成功
- 记录密码修改日志
