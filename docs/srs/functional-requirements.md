# 3 功能需求（需求必须编号，每项需求内参考作业文档中基本描述、相关功能的结构，并根据需要增减）

## 3.1 企业

| 需求标识符   | FR-101                                                       |
| :----------- | :----------------------------------------------------------- |
| **需求名称** | 数据填报与暂存                                               |
| **基本描述** | 企业用户通过系统进行基础数据的在线填报，并支持对未提交的数据进行临时保存。 |
| **相关功能** | 数据录入、数据编辑、数据暂存、状态标识                       |

| 需求标识符   | FR-102                                                       |
| :----------- | :----------------------------------------------------------- |
| **需求名称** | 数据提交与状态跟踪                                           |
| **基本描述** | 企业用户完成填报后可将数据正式提交至所属市级单位，并可查看已上报数据的审核状态及历史记录。 |
| **相关功能** | 数据提交、进度查询、审核状态查看                             |

| 需求标识符   | FR-103                                                       |
| :----------- | :----------------------------------------------------------- |
| **需求名称** | 退回数据处理与再上报                                         |
| **基本描述** | 企业用户可查看市级单位退回的数据及退回意见，并根据意见对数据进行修改后重新提交。 |
| **相关功能** | 退回意见查看、数据修改、重新上报                             |

## 3.3 市

| 需求标识符   | FR-301                                                       |
| :----------- | :----------------------------------------------------------- |
| **需求名称** | 企业数据审核                                                 |
| **基本描述** | 市级用户对辖区内企业提交的数据进行完整性、规范性核验，支持逐条或批量审核操作。 |
| **相关功能** | 待审核列表、数据查看、逐条审核、批量审核                     |

| 需求标识符   | FR-302                                                       |
| :----------- | :----------------------------------------------------------- |
| **需求名称** | 数据退回与意见反馈                                           |
| **基本描述** | 对于审核不通过的数据，市级用户需填写明确的退回意见，并通过系统将数据退回至对应企业。 |
| **相关功能** | 退回操作、退回意见填写、企业通知                             |

| 需求标识符   | FR-303                                                       |
| :----------- | :----------------------------------------------------------- |
| **需求名称** | 市级数据汇总与上报                                           |
| **基本描述** | 市级用户对审核通过的数据进行自动汇总，并在确认全市数据无误后，执行上报省级操作。 |
| **相关功能** | 数据汇总、汇总数据查看、上报省级                             |

| 需求标识符   | FR-304                                                       |
| :----------- | :----------------------------------------------------------- |
| **需求名称** | 上报数据更正申请                                             |
| **基本描述** | 数据上报省级后但在省级审核前，若发现错误，市级用户可申请对已上报数据进行退回或更正。 |
| **相关功能** | 退回申请、状态锁定、更正提交                                 |
