# 3 功能需求（需求必须编号，每项需求内参考作业文档中基本描述、相关功能的结构，并根据需要增减）

## 3.1 企业

**FR-101 企业信息修改**
企业用户可修改本企业的基础信息。修改完成后需保存并上报至省局备案。企业信息需按照统一模板填写，部分字段不可修改（如所属地区、组织机构代码等）。
**相关功能：**

- 查看当前企业信息
- 编辑可修改字段
- 保存修改并触发备案流程

**FR-102 备案上报**
企业完成信息填写或修改后，可将备案信息上报至省局。上报后信息进入待审核状态。
**相关功能：**

- 提交备案信息
- 提示提交成功
- 查看备案状态（待审核/已通过/已退回）

**FR-103 数据填报**
企业在规定的上报期内，填报本企业的就业失业数据。填报内容需符合模板要求，若调查期就业人数低于建档期人数，则需填写减少类型及原因。
**相关功能：**

- 填报当期数据
- 保存草稿
- 提交上报
- 校验必填项及逻辑规则

**FR-104 数据查询**
企业用户可查询历史填报数据及其状态（如已上报、已审核、已退回等）。查询结果仅支持浏览，不支持导出。
**相关功能：**

- 按调查期查询
- 查看数据详情
- 查看审核状态及退回备注

**FR-105 密码修改**
企业用户可修改登录密码。修改后需重新登录。
**相关功能：**

- 验证旧密码
- 输入新密码并确认
- 提示修改成功

## 3.3 市

**FR-301 数据审核**
市级用户审核辖区内企业上报的就业失业数据。审核通过后数据可上报至省级。
**相关功能：**

- 列表展示待审核数据
- 查看企业填报详情
- 审核通过/退回（可填写备注）
- 查看审核记录

**FR-302 通知发布**
市级用户可向辖区内企业发布通知。通知内容需包含标题、正文、发布时间等信息。
**相关功能：**

- 新增通知（标题、内容）
- 修改通知
- 删除通知
- 查看已发布通知列表

**FR-303 密码修改**
市级用户可修改登录密码。修改后需重新登录。
**相关功能：**

- 验证旧密码
- 输入新密码并确认
- 提示修改成功
